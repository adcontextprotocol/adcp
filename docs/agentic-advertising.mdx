# What is agentic advertising?

Agentic advertising is the practice of using AI agents to autonomously execute advertising tasks — media planning, audience targeting, creative generation, campaign execution, and performance optimization — on behalf of advertisers. Instead of humans manually operating each advertising platform, AI agents handle the execution while humans set strategy and approve decisions.

## How it works

In agentic advertising, specialized AI agents take on roles that were previously manual:

- A **media buying agent** discovers inventory across publishers, evaluates proposals, and executes buys
- A **creative agent** generates ad creatives from brand guidelines and product catalogs (sales agents often fill this role too — see [creative capabilities on sales agents](/docs/creative/sales-agent-creative-capabilities))
- A **signals agent** identifies and activates audience segments for targeting
- A **governance agent** enforces brand safety rules and content standards
- An **orchestrator** coordinates these agents to run end-to-end campaigns

These agents communicate through protocols — structured interfaces that define how they discover capabilities, exchange data, and execute transactions.

## Why it matters

Advertising operations are complex and fragmented. A mid-size advertiser might work with 10-15 platforms, each with its own API, dashboard, and workflow. Scaling to more partners means scaling headcount.

Agentic advertising changes this equation. An AI agent using a standard protocol can interact with any platform that implements that protocol. One integration gives access to every compatible partner.

The key requirements:

| Requirement | Why it matters |
|-------------|---------------|
| **Standard protocol** | Agents need a shared language to communicate across platforms |
| **Structured tasks** | Operations must be well-defined enough for autonomous execution |
| **Human oversight** | Strategy and approval remain with humans; agents handle execution |
| **Platform agnostic** | The same agent should work with any advertising platform |

## AdCP: the open standard for agentic advertising

The Ad Context Protocol (AdCP) is the open standard that makes agentic advertising work. Published by [AgenticAdvertising.org](https://agenticadvertising.org), AdCP defines the tasks, schemas, and transport mechanisms that AI agents use to participate in advertising workflows.

AdCP covers eight domains:

- **[Media Buy](/docs/media-buy)** — Inventory discovery, campaign execution, delivery reporting
- **[Creative](/docs/creative)** — Creative generation, format discovery, AI provenance
- **[Signals](/docs/signals/overview)** — Audience and contextual data activation
- **[Governance](/docs/governance/overview)** — Brand safety, content standards, compliance
- **[Accounts](/docs/accounts/overview)** — Billing relationships and financial reporting
- **[Brand Protocol](/docs/brand-protocol)** — Machine-readable brand identity via `brand.json`
- **[Sponsored Intelligence](/docs/sponsored-intelligence/overview)** — Conversational brand experiences
- **[Registry](/docs/registry)** — Agent and brand discovery

AdCP works over two transport protocols:
- **MCP (Model Context Protocol)** for direct tool-calling from AI assistants
- **A2A (Agent-to-Agent Protocol)** for multi-agent collaboration

## How agentic advertising differs from programmatic

Programmatic advertising automates the *transaction* — real-time bidding on individual impressions. Agentic advertising automates the *workflow* — the entire process from strategy to execution.

| | Programmatic | Agentic |
|---|---|---|
| **What's automated** | Bid decisions at impression level | End-to-end campaign workflows |
| **Human role** | Configure platform settings | Set strategy, approve plans |
| **Integration model** | Platform-specific APIs | Standard protocol (AdCP) |
| **Scope** | Media buying | Media, creative, audiences, governance, accounts |
| **Communication** | Machine-to-machine (OpenRTB) | Agent-to-agent (AdCP over MCP/A2A) |

Agentic advertising builds on programmatic infrastructure. AdCP works alongside existing standards like OpenRTB — it doesn't replace them. Instead, it provides the agent-level coordination layer that programmatic systems lack.

## Frequently asked questions

<AccordionGroup>
  <Accordion title="What protocols do AI advertising agents use?">
    AI advertising agents use the Ad Context Protocol (AdCP), an open standard published by AgenticAdvertising.org. AdCP defines structured tasks for media buying, creative generation, audience targeting, and governance that work over MCP (Model Context Protocol) and A2A (Agent-to-Agent Protocol) transports.
  </Accordion>

  <Accordion title="How does AdCP relate to IAB standards like OpenRTB?">
    AdCP complements IAB standards rather than replacing them. OpenRTB defines how advertising systems communicate at the impression level (bid requests and responses). AdCP defines how AI agents coordinate at the workflow level (discovering inventory, executing campaigns, managing creatives). An AdCP agent might use a platform that participates in OpenRTB auctions under the hood.
  </Accordion>

  <Accordion title="What is AgenticAdvertising.org?">
    AgenticAdvertising.org is the member organization that stewards the Ad Context Protocol (AdCP) and related open standards for AI-powered advertising. Members include publishers, platforms, agencies, brands, and technology providers working to standardize how AI agents interact in advertising.
  </Accordion>

  <Accordion title="Do I need to be a developer to use agentic advertising?">
    No. Tools like [Addie](https://adcontextprotocol.org/chat) let you explore agentic advertising through conversation — no code required. Through vibe coding — describing what you want in plain language — anyone can build a working advertising agent, no programming experience required. The [certification program](/docs/learning/overview) walks you through it. For technical integration, AdCP provides [client libraries](/docs/intro#client-libraries) in TypeScript and Python.
  </Accordion>

  <Accordion title="Is agentic advertising the same as AI in advertising?">
    Not exactly. AI in advertising is broad — it includes recommendation algorithms, bid optimization, audience modeling, and creative testing. Agentic advertising specifically refers to AI agents that autonomously execute advertising workflows on behalf of humans, using protocols like AdCP to communicate across platforms.
  </Accordion>
</AccordionGroup>
