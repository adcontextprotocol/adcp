# Platform / intermediary track (D1–D3)

<Info>
**Members only** — Requires A3 completion. Three modules, ~39 minutes total.
</Info>

This track is for people building AdCP infrastructure: ad tech platforms, exchanges, data companies, and anyone connecting the ecosystem. You'll learn MCP server architecture, supply chain verification, and how to migrate existing programmatic infrastructure to AdCP.

Completing this track (plus A1–A3) earns the **AdCP practitioner** credential.

---
