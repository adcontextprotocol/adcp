# Specialist capstone — Media buy (E1)

<Info>
**Members only** — Requires Practitioner credential. ~30 minutes with Addie. Combines hands-on lab and adaptive exam.
</Info>

This capstone tests your mastery of the media buy transaction lifecycle. You'll work with live sandbox agents to execute the complete flow: product discovery, campaign creation, modification, and delivery reporting. Addie evaluates both your hands-on work and your conceptual understanding.

Passing earns the **AdCP specialist — Media buy** credential.

## What you'll demonstrate

- Execute a complete media buy transaction against sandbox agents
- Handle pricing negotiation, budget allocation, and multi-agent orchestration
- Monitor and optimize campaign delivery using protocol tools
- Reason about failure modes, conflict resolution, and edge cases

## Lab exercises

During the capstone, Addie will guide you through hands-on exercises:

1. **Product discovery and evaluation** — Query multiple sandbox agents, compare products, evaluate pricing
2. **Campaign creation** — Create a media buy with specific targeting, budget, and creative requirements
3. **Campaign optimization** — Monitor delivery, identify underperformance, execute updates
4. **Multi-agent orchestration** — Manage campaigns across multiple sellers simultaneously

## Assessment

| Dimension | Weight | What Addie evaluates |
|-----------|--------|---------------------|
| Protocol mastery | 30% | Correct use of media buy tasks and schemas |
| Transaction flow | 25% | Can you execute the full lifecycle without guidance? |
| Problem solving | 20% | How do you handle errors, conflicts, and edge cases? |
| Optimization reasoning | 15% | Can you interpret delivery data and make informed decisions? |
| Communication | 10% | Can you explain your decisions and reasoning clearly? |

Passing threshold: 70%.
